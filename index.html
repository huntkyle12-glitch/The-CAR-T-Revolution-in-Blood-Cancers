Background

What We're Looking At

Hematologic cancers are cancers of the blood, bone marrow, and lymphatic system. For decades, treatment options were limited — but cellular immunotherapy has changed the equation.

🩸

Hematologic Malignancies

Cancers of the blood-forming tissues — leukemias (ALL, AML, CLL), lymphomas (DLBCL, Hodgkin), and multiple myeloma — account for roughly 10% of all new U.S. cancer diagnoses. Their surface antigen expression makes them uniquely well-suited for targeted immunotherapy.

⚗️

CAR-T Cell Therapy

Chimeric Antigen Receptor T-cell therapy engineers a patient's own immune cells to recognize and destroy cancer. T-cells are extracted, modified to express a receptor targeting a tumor antigen (CD19, BCMA), expanded ex vivo, and reinfused. FDA first approved this class of therapy in August 2017.

📊

The Data Story

This project traces two parallel trends: slowly rising incidence of blood cancers alongside dramatic improvement in 5-year survival that corresponds closely with the arrival of CAR-T and novel targeted therapies. Data sourced from NCI SEER, ClinicalTrials.gov, and the FDA.

🔬

Why This Matters

Understanding the relationship between trial activity and downstream survival outcomes is central to evaluating drug pipelines. With myeloma 5-year survival improving nearly 26 pp since 2001, these trends offer compelling proof of concept for the cell and gene therapy era underway.

Analytical Pipeline · SQL · Python · Tableau

How the Data Was Processed

This project follows a reproducible three-stage workflow: SQL extraction from structured SEER and ClinicalTrials.gov database exports, Pandas cleaning and transformation, then Tableau for interactive dashboarding and publication.

01
SQL — ExtractQuery structured SEER and ClinicalTrials.gov exports in PostgreSQL. Filter by cancer site, diagnosis year, and trial keyword. Aggregate survival and incidence metrics.
02
Python / Pandas — TransformLoad query outputs into DataFrames. Normalize rates, reshape long→wide, compute period-over-period deltas and z-scores, export clean CSVs.
03
Tableau — VisualizeConnect cleaned CSVs as flat-file data sources. Build survival trend lines, trial growth charts, and CAR-T milestone annotation layers for a Tableau Public dashboard.
queries/seer_survival_extract.sql PostgreSQL 
-- ─────────────────────────────────────────────────────────────────────
-- NCI SEER + ClinicalTrials.gov extraction  |  Kyle Hunt  |  2025
-- Tables loaded from SEER*Stat ASCII exports and CT.gov bulk download
-- ─────────────────────────────────────────────────────────────────────

-- 1. Five-year relative survival by cancer type and diagnosis year
SELECT
    s.cancer_site_recode                             AS cancer_type,
    s.diagnosis_year,
    COUNT(s.patient_id)                             AS case_count,
    ROUND(AVG(s.relative_survival_5yr) * 100, 1)  AS survival_pct
FROM   seer.survival_records s
WHERE  s.cancer_site_recode IN (
           'Non-Hodgkin Lymphoma', 'Multiple Myeloma',
           'Acute Lymphoblastic Leukemia',
           'Acute Myeloid Leukemia', 'Chronic Lymphocytic Leukemia'
       )
  AND  s.diagnosis_year  BETWEEN 2001 AND 2021
  AND  s.follow_up_months >= 60          -- require 5-yr follow-up complete
  AND  s.vital_status_recode IS NOT NULL
GROUP BY  s.cancer_site_recode, s.diagnosis_year
ORDER BY  s.cancer_site_recode, s.diagnosis_year;


-- 2. Age-adjusted incidence rates (SEER case-level + population tables)
SELECT
    i.cancer_site_recode                             AS cancer_type,
    i.diagnosis_year,
    SUM(i.case_count)                               AS raw_cases,
    ir.age_adjusted_rate                             -- pre-computed in SEER export
FROM   seer.incidence_records   i
JOIN   seer.age_adjusted_rates  ir
         ON  ir.cancer_site = i.cancer_site_recode
         AND ir.year        = i.diagnosis_year
WHERE  i.diagnosis_year BETWEEN 2000 AND 2021
GROUP BY  i.cancer_site_recode, i.diagnosis_year, ir.age_adjusted_rate
ORDER BY  i.cancer_site_recode, i.diagnosis_year;


-- 3. CAR-T trial registrations per year (ClinicalTrials.gov bulk dump)
SELECT
    EXTRACT(YEAR FROM ct.start_date)               AS reg_year,
    COUNT(ct.nct_id)                                AS trial_count,
    SUM(COUNT(ct.nct_id))
        OVER (ORDER BY EXTRACT(YEAR FROM ct.start_date))
                                                     AS cumulative_trials
FROM   clinicaltrials.studies ct
WHERE (
           LOWER(ct.official_title) LIKE '%chimeric antigen receptor%'
        OR LOWER(ct.official_title) LIKE '%car-t%'
        OR LOWER(ct.official_title) LIKE '%car t cell%'
       )
  AND  ct.condition_mesh LIKE '%hematolog%'
  AND  ct.start_date BETWEEN '2010-01-01' AND '2023-12-31'
GROUP BY  reg_year
ORDER BY  reg_year;
notebooks/transform_seer_data.py Python 3.11 · Pandas · NumPy 
import pandas  as pd
import numpy   as np
from  pathlib  import Path

DATA_DIR = Path("data/raw")
OUT_DIR  = Path("data/clean")
OUT_DIR.mkdir(exist_ok=True)

# ── 1. Load SQL exports ────────────────────────────────────────────
survival_raw  = pd.read_csv(DATA_DIR / "seer_survival_extract.csv")
incidence_raw = pd.read_csv(DATA_DIR / "seer_incidence_extract.csv")
trials_raw    = pd.read_csv(DATA_DIR / "clinicaltrials_cart_counts.csv")

# ── 2. Survival — clean, type-cast, compute deltas ────────────────
df_surv = (
    survival_raw
    .rename(columns={"cancer_type": "cancer", "survival_pct": "rate"})
    .dropna(subset=["rate"])
    .astype({"diagnosis_year": "int16", "rate": "float32"})
    .query("2001 <= diagnosis_year <= 2021")
)

# Pivot long → wide (one column per cancer type for Tableau)
surv_wide = (
    df_surv
    .pivot_table(
        values  = "rate",
        index   = "diagnosis_year",
        columns = "cancer",
        aggfunc = "mean"
    )
    .reset_index()
    .round(1)
)

# Compute pp delta vs 2001 baseline for each cancer type
baseline = surv_wide[surv_wide["diagnosis_year"] == 2001].iloc[0]
for col in surv_wide.columns[1:]:
    surv_wide[f"{col}_delta_pp"] = (surv_wide[col] - baseline[col]).round(1)

# ── 3. Incidence — normalize, add era flag ────────────────────────
df_inc = (
    incidence_raw
    .rename(columns={"age_adjusted_rate": "rate_per_100k"})
    .assign(
        cart_era = lambda x: np.where(x["diagnosis_year"] >= 2017,
                                          "Post-CAR-T", "Pre-CAR-T"),
        rate_z   = lambda x: (          # z-score for cross-type comparison
            (x["rate_per_100k"] - x["rate_per_100k"].mean())
            / x["rate_per_100k"].std()
        )
    )
)

# ── 4. Trials — compute YoY growth and doubling time ──────────────
df_trials = (
    trials_raw
    .sort_values("reg_year")
    .assign(
        yoy_pct  = lambda x: x["trial_count"].pct_change() * 100,
        doubling = lambda x:
            np.log(2) / np.log(1 + x["yoy_pct"] / 100)
                        .replace([np.inf, -np.inf], np.nan)
    )
)

# Quick sanity check
print(df_trials[["reg_year", "trial_count", "cumulative_trials", "yoy_pct"]])
print(f"\nMax YoY growth: {df_trials['yoy_pct'].max():.1f}%")
print(f"Total 2010→2023 growth: {df_trials['cumulative_trials'].iloc[-1] / df_trials['cumulative_trials'].iloc[0]:.0f}x")

# ── 5. Export clean CSVs for Tableau ─────────────────────────────
surv_wide.to_csv(OUT_DIR / "survival_wide.csv",   index=False)
df_inc.to_csv(OUT_DIR    / "incidence_clean.csv", index=False)
df_trials.to_csv(OUT_DIR / "trials_growth.csv",   index=False)
print("✓ All exports complete — ready for Tableau ingestion.")
